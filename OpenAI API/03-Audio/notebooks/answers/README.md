# Respostas dos Exercicios - Audio

Este diretorio contem as solucoes estruturadas para os exercicios de processamento de audio (TTS e STT).

---

## Solucoes por Tutorial

### Tutorial 01: Text-to-Speech (TTS)
- [tut-01-ex-01-narracao-artigo.py](tut-01-ex-01-narracao-artigo.py): Conversao de textos longos para audio de alta fidelidade.

### Tutorial 02: Speech-to-Text (Whisper)
- [tut-02-ex-02-detector-idioma.py](tut-02-ex-02-detector-idioma.py): Extracao automatica de metadados e deteccao de lingua original.

### Tutorial 03: Traducao de Audio
- [tut-03-ex-02-pipeline-multilingue.py](tut-03-ex-02-pipeline-multilingue.py): Fluxo hibrido de transcricao e traducao para multiplos idiomas alvo.

---

## Como Executar

1. Instale as dependencias:
```bash
pip install openai python-dotenv
```
2. Configure sua `OPENAI_API_KEY`.
3. Execute o script:
```bash
python tut-01-ex-01-narracao-artigo.py
```
