# Respostas dos Exercícios - Advanced

Este diretório contém as respostas resolvidas dos exercícios do tutorial de Extração de Dados JSON.

## Tutorial 01: Extração de Dados JSON
- `tut-01-ex-01-extrair-dados-email.py` - Extração de dados de e-mail estruturado
- `tut-01-ex-02-processar-multiplos-documentos.py` - Processamento de múltiplos PDFs
- `tut-01-ex-03-validacao-tratamento-erros.py` - Validação robusta com JSON Schema e retry

## Como Executar
1. Certifique-se de ter o arquivo `.env` configurado.
2. Instale as dependências: `pip install openai python-dotenv pypdf jsonschema`
3. Execute: `python tut-01-ex-01-extrair-dados-email.py`
